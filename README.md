# Turnigy Absorber Load

Absorber load is currently a Turnigy SK8 6374-149KV ($119.99):

 * 80A max current, 3500W
 * Resistance 21mOhm (VESC reports ~18mOhm)
 * 8-12s LiPO rated
 * 8mm shaft
 * 14 pole pairs

VESC determined data (12/19/2024):

 * 18.5mOhm
 * 11.34uH
 * Hall sensors
 * Detuned to +/- 20A current
